# Akin

Akin is a privacy-focused transport support platform for campus communities.

It provides a Go API and a React PWA for account access, role-aware user flows, payments, ride coordination, safety actions, and internal review tools.

## Stack

- Go, Fiber, GORM, Postgres, Redis
- Vite, React, TypeScript, Tailwind
- Paystack payments
- Resend email
